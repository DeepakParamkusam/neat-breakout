# neat-breakout
Implementation of NEAT algorithm on Breakout clone Bricka.  
  
Dependencies-  
1. neat-python v0.7 by CodeReclaimers <https://github.com/CodeReclaimers/neat-python>  
2. pygame v1.9.3
